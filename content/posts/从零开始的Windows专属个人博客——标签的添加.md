# 从零开始的Windows专属个人博客——标签的添加

> 你是不是看着别人的博客，文章下面飘着一串五彩斑斓的标签，心里羡慕得不行？别急，今天咱们就来给你的博客也"贴贴标签"，让它看起来更专业、更有范儿！

## 一、标签是个啥？能吃吗？

标签（Tags），说白了就是给你的文章"分类打标记"。就像超市里的商品贴着"特价"、"进口"、"无糖"的标签一样，你的文章也可以贴上"Git"、"教程"、"踩坑记录"这些标签。

**标签的好处**：

| 功能 | 说明 |
|------|------|
| 快速筛选 | 点击标签，一键找到所有相关文章 |
| 视觉美观 | 文章下方飘着一串标签，看着就很专业 |
| SEO优化 | 搜索引擎更喜欢有标签的网站 |

## 二、PaperMod主题：标签功能的"原生玩家"

好消息！你用的 **PaperMod 主题**天生就支持标签功能，不需要你写任何代码，只需要在文章里"打个标记"就行。

**PaperMod 自动帮你做的事**：

- 📋 自动生成 `/tags/` 页面（标签云）
- 🏷️ 文章列表页显示标签
- 📄 文章详情页显示标签
- 🔍 点击标签自动筛选相关文章

## 三、实操时间：给你的文章"贴标签"

### 步骤1：打开文章文件

找到你的文章文件，比如 `content/posts/我的第一篇博客.md`。

用记事本、VS Code、或者任何编辑器打开它。

### 步骤2：找到文章开头的那几行

文章开头有一段"神秘代码"，叫做 **Front Matter**（前言）。看起来像这样：

```yaml
---
title: "我的第一篇博客"
date: 2026-06-28T10:00:00+08:00
draft: false
---
```

> 这段代码告诉 Hugo：文章标题是啥、发布时间是啥、是不是草稿状态。

### 步骤3：加上 tags 字段

在 Front Matter 里加上 `tags` 字段，就像这样：

```yaml
---
title: "我的第一篇博客"
date: 2026-06-28T10:00:00+08:00
draft: false
tags: ["Git", "教程", "踩坑"]
---
```

> 注意：标签写在方括号 `[]` 里，用双引号 `""` 包裹，多个标签用逗号 `,` 分隔。

### 步骤4：保存文件，推送代码

保存文章文件，然后执行老三样：

```powershell
git add .
git commit -m "给文章添加标签"
git push
```

等待 GitHub Actions 自动部署完成（1-3分钟），刷新网页就能看到标签啦！

## 四、标签命名小技巧

### ✅ 推荐做法

| 标签类型 | 示例 |
|----------|------|
| 技术名词 | `Git`, `Python`, `Hugo` |
| 文章类型 | `教程`, `笔记`, `踩坑` |
| 难度级别 | `入门`, `进阶`, `高级` |

### ❌ 不推荐做法

| 错误示范 | 问题 |
|----------|------|
| `git教程踩坑` | 太长，不好点击 |
| `超级无敌难` | 太主观，别人看不懂 |
| `文章1` | 没意义，不如不贴 |

## 五、常见问题 Q&A

### Q1：标签和分类有啥区别？

| 功能 | 标签 (Tags) | 分类 (Categories) |
|------|-------------|-------------------|
| 数量 | 一篇文章可以有多个标签 | 一篇文章通常只有一个分类 |
| 层级 | 无层级，平铺展示 | 可以有层级（如：技术 → 编程 → Python） |
| 用途 | 详细标记文章主题 | 大范围归类文章 |

**简单理解**：分类是"大箱子"，标签是"小贴纸"。

### Q2：标签页在哪里？

访问 `https://你的用户名.github.io/tags/` 就能看到所有标签的集合页面！

### Q3：标签太多会不会乱？

建议每篇文章 **2-5 个标签**，太多反而让读者眼花缭乱。

### Q4：标签颜色能改吗？

PaperMod 主题的标签颜色由主题控制，如果想自定义，需要修改 CSS 文件。这个咱们后面专门讲。

## 六、实战案例：给四篇文章添加标签

让我带你实战一下，给四篇不同类型的文章添加合适的标签：

### 文章1：Git操作手册

```yaml
---
title: "Git 完整操作手册"
date: 2024-06-21T17:00:00+08:00
draft: false
tags: ["Git", "工具", "教程"]
---
```

> 这是一篇Git教程，贴上"Git"、"工具"、"教程"三个标签，清晰明了。

### 文章2：Hugo博客搭建教程

```yaml
---
title: "从零开始的Windows专属个人博客搭建全攻略"
date: 2026-06-20T12:00:00+08:00
draft: false
tags: ["Hugo", "博客", "教程", "GitHub"]
---
```

> 搭建教程类文章，贴上"Hugo"、"博客"、"教程"、"GitHub"四个标签。

### 文章3：主题样式指南

```yaml
---
title: "博客主题样式操作指南"
date: 2024-06-21T16:00:00+08:00
draft: false
tags: ["PaperMod", "样式", "配置", "指南"]
---
```

> 主题配置类文章，贴上"PaperMod"、"样式"、"配置"、"指南"四个标签。

### 文章4：测试笔记

```yaml
---
title: "我的第一篇笔记"
date: 2024-06-21T15:30:00+08:00
draft: false
tags: ["测试", "笔记"]
---
```

> 测试类文章，简单贴两个标签就行。

## 七、标签页面预览效果

添加标签后，你的博客会出现这些变化：

| 页面 | 效果 |
|------|------|
| 文章列表 | 每篇文章下方显示标签小字 |
| 文章详情 | 标题下方显示标签列表 |
| 标签页 `/tags/` | 所有标签云展示，点击可筛选 |
| 导航栏 | "标签"菜单自动生成 |

## 八、总结

标签功能就这么简单！总结一下核心步骤：

```
打开文章 → 找到Front Matter → 添加tags字段 → 保存推送 → 完成
```

**一句话概括**：三分钟搞定，让你的博客从"裸奔"变成"有标签的专业选手"！
