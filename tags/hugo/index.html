从零开始的Windows专属个人博客搭建全攻略（GitHub Pages+Hugo PaperMod）

保姆级教程：从零搭建免费Hugo PaperMod个人博客（GitHub永久部署） 手把手教小白搭建免费、无广告、高颜值、可自动更新的个人博客，基于 Hugo + PaperMod 主题 + GitHub 全自动部署，无需服务器、无需付费，一次搭建永久使用。 一、核心原理：先搞懂整套博客运行逻辑 整套博客分为本地编写和云端部署两大模块，零基础也能轻松理解： 本地电脑：安装工具、撰写博客文章、实时预览网页效果 GitHub云端：自动打包网页代码、生成公开专属网址，全网可访问 三件核心工具（人话翻译）： Git：文件传输工具，负责将本地博客文件同步到GitHub云端仓库（云端U盘） Hugo Extended：静态网页生成工具，将纯文本笔记转换成精美网页 GitHub：免费代码托管平台，实现网页自动构建、部署、公网访问 二、环境准备：电脑必备软件安装（小白零难度） 1. 安装 Git（代码传输工具） 浏览器搜索「Git官网」，下载 Windows 正式版安装包 全程默认安装，一路点击 Next，无需修改任何配置 安装成功标识：桌面/文件夹右键，出现 Git Bash Here、Git GUI Here 选项 小贴士：命令行无需手动敲代码，全程复制粘贴指令即可，不用死记命令！ 2. 安装 Hugo Extended 扩展版（核心必装） 重点提醒：必须安装 Extended 扩展版，普通版缺失主题功能，会导致博客报错！ 打开 Hugo 官方发布页，下载 Windows 端 .zip 压缩包 解压压缩包，将内部 hugo.exe 单独放入纯英文无中文路径文件夹，示例：F:\hugo 将该文件夹添加到电脑系统环境变量，添加完成后重启 Git Bash 验证安装：右键打开 Git Bash，输入 hugo version，输出版本号即安装成功 3. 注册 GitHub 账号并创建仓库 GitHub 官网邮箱注册账号，用户名建议简短好记（后续仓库名必须和用户名完全一致） 新建公开仓库，仓库名称严格遵循固定格式：你的用户名.github.io 示例：GitHub 用户名为 xiaobai123，仓库名必须为 xiaobai123.github.io，名称错误会直接导致免费域名失效！ ...
