从零开始的Windows专属个人博客——进阶功能解锁

从零开始的Windows专属个人博客——进阶功能解锁 上一篇我们学会了给文章"贴标签"，今天咱们继续升级！解锁三个超级实用的功能：🌙 主题切换、⬆️ 返回顶部、🎮 404小游戏。让你的博客从"能用"变成"好用又好玩"！ 一、🌙 主题切换：白天黑夜一键切换 为什么需要深色模式？ 想象一下：你在深夜加班，打开博客看教程，结果白花花的页面刺得眼睛生疼…这时候深色模式就像深夜里的一盏小台灯，温柔又护眼。 开启方法 打开 hugo.yml，找到这两行： params: disableThemeToggle: false # 改为 false 开启切换 defaultTheme: auto # auto 自动跟随系统，light 始终浅色，dark 始终深色 配置说明： 参数 值 效果 disableThemeToggle false 开启主题切换按钮 disableThemeToggle true 禁用主题切换按钮 defaultTheme auto 跟随系统设置 defaultTheme light 默认浅色模式 defaultTheme dark 默认深色模式 效果预览 开启后，导航栏右侧会出现一个 🌙/☀️ 图标，点击即可切换主题！ 二、⬆️ 返回顶部：一键直达 为什么需要返回顶部？ 你有没有这样的经历：读一篇很长的文章，翻到末尾想回到开头，结果手指在屏幕上疯狂滑动？返回顶部按钮就是你的"时光机"，一键回到页面顶部！ 开启方法 同样在 hugo.yml 中： params: disableScrollToTop: false # 改为 false 开启返回顶部按钮 效果预览 当你向下滚动页面超过一定距离时，右下角会出现一个可爱的 ↑ 按钮，点击即可平滑回到顶部。 三、🎮 404小游戏：迷路也能玩一会儿 为什么要做有趣的404页面？ 用户访问了一个不存在的页面，本来应该很沮丧。但如果404页面上有一个好玩的小游戏，用户可能会忘记刚才的"迷路"，反而觉得你的博客很有趣！ 我们做了什么？ 我们创建了一个 点击笑脸得分游戏： 🎯 游戏规则：30秒内点击随机出现的笑脸得分 ⚡ 难度递增：时间越短，笑脸移动越快 ✨ 特效反馈：点击时有爆炸特效 🏆 结算界面：游戏结束后显示最终得分 实现原理 这是一个纯前端游戏，不需要任何后端支持： ...
